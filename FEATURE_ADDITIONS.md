### 1. QR Code Generation (5 min to implement)
Every event and share link gets a QR code. Uses `qrcode` npm package.
- Show on event detail page
- Include in share dialog
- Downloadable as PNG
- Scannable for event check-in

```bash
npm install qrcode @types/qrcode
```

```typescript
// lib/utils/qrcode.ts
import QRCode from 'qrcode'

export async function generateQRCode(url: string): Promise<string> {
  return QRCode.toDataURL(url, {
    width: 256,
    margin: 2,
    color: { dark: '#000000', light: '#FFFFFF' },
  })
}
```
